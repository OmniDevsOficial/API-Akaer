# 🚀 Backend - API Akaer

## 📌 Descrição
Back-end da plataforma de normas aeronáuticas, responsável pela autenticação de usuários.

- 🔐 Criptografia de senha com bcrypt
- 🎟️ Autenticação com JWT
- 🧠 Validação de senha

---

## 🛠️ Tecnologias
- Node.js
- TypeScript
- Express (estrutura base)
- Prisma ORM
- SQLite (dev)
- Bcrypt
- JWT
